**Syntax**

```
<!--{ inline comment here }-->
	
<!--{ 
	
	Multi-line comment here 
	This multiline comment is not sent to browser.
	
}-->
```

**Example**

```
<h1>Hello world</h1>

<!--{ This is a comment }-->
<span>Powered by Div</span>
```

**Output**

```

<h1>Hello world</h1>
<span>Powered by Div</span>
```
